  🧠 Overview

Puzzle Shuffle Game is a modern, lightweight, and interactive browser game built with pure frontend technologies.
Designed with smooth animations and a minimal UI, this project focuses on user interaction, visual feedback, and cognitive engagement.

💡 Easy to play. Hard to master.

✨ Key Highlights
🎯 Interactive Gameplay – Click & hover-based puzzle interactions
⚡ Smooth Animations – Scale, glow, and flip effects
🧩 Grid System – Structured 4×4 puzzle layout
🌙 Dark UI Theme – Clean and distraction-free design
🚀 Zero Dependencies – Pure HTML, CSS & JavaScript
🎮 Gameplay Experience
Hover over tiles → feel responsive scaling & glow
Click tiles → trigger flip interaction
Analyze patterns → solve the puzzle efficiently

🧠 This game improves memory, focus, and observation skills.
